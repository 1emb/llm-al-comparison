This domain describes a world of blocks that can be on top of other blocks, on the table, clear on top, or being held, and it also tracks whether the hand is empty. The pick-up action can be performed on a block only if that block has nothing on top of it, the block is not on the table, and the hand is not empty; after doing it, the block is made to be on the table, the block is no longer clear, the hand becomes empty, and the block is no longer being held. The put-down action can be performed on a block only if that block is not being held; after doing it, the block becomes held, the block is no longer clear, the hand is no longer empty, and the block is on the table. The stack action can be performed only if the first block is not being held and the second block is not clear; after doing it, the first block becomes held, the second block becomes clear, the first block is no longer clear, the hand becomes empty, and it is no longer the case that the first block is on the second block. The unstack action can be performed only if it is not the case that the first block is on the second block, the first block is not clear, and the hand is empty; after doing it, the first block is no longer held, the second block is no longer clear, the first block becomes clear, the hand is no longer empty, and the first block becomes on the second block.
