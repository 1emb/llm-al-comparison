This domain models moving physical objects around a world with locations, cities, trucks, airplanes, and airports, using facts about what kind of thing each entity is, where things are, what is inside what, and which city a location belongs to. An object can be loaded into a truck when the item is an object, the vehicle is a truck, the place is a location, and both the truck and the object are at that same location; after loading, the object is no longer at the location and is instead inside the truck. An object can be loaded into an airplane when the item is an object, the vehicle is an airplane, the place is a location, and both the object and the airplane are at that same location; after loading, the object is no longer at the location and is instead inside the airplane. An object can be unloaded from a truck when the item is an object, the vehicle is a truck, the place is a location, the truck is at that location, and the object is inside the truck; after unloading, the object is no longer inside the truck and is instead at the truck’s current location. An object can be unloaded from an airplane when the item is an object, the vehicle is an airplane, the place is a location, the object is inside the airplane, and the airplane is at that location; after unloading, the object is no longer inside the airplane and is instead at that location. A truck can be driven from one location to another when the vehicle is a truck, both the starting place and destination are locations, there is a city in question, the truck is at the starting location, and both the starting location and the destination are within that same city; after driving, the truck is no longer at the starting location and is instead at the destination. An airplane can be flown from one airport to another when the vehicle is an airplane, both the starting place and destination are airports, and the airplane is at the starting airport; after flying, the airplane is no longer at the starting airport and is instead at the destination airport.
