This domain describes a world of blocks where a block can be on another block, on the table, clear (with nothing on top of it), held in a hand, and where the hand can be empty. The pick-up action can occur when the chosen block is clear, the hand is empty, and it is not on the table; after doing it, the block is no longer on the table, the block is marked clear, the hand is marked empty, and the hand is also marked as holding that block. The put-down action can occur when it is not the case that the hand is holding the chosen block; after doing it, the state is changed so that the hand is no longer holding that block, the block is no longer clear, the hand becomes empty, and the block is no longer on the table. The stack action can occur when it is not the case that the hand is holding the block to be stacked and the destination block is clear; after doing it, the hand is no longer holding the moved block, the destination block is no longer clear, the moved block is no longer clear, the hand becomes empty, and it is no longer the case that the moved block is on the destination block. The unstack action can occur when it is not the case that one block is on the other, the first block is clear, and the hand is empty; after doing it, the hand is no longer holding the first block, the second block becomes clear, the first block is no longer clear, the hand is no longer empty, and it becomes the case that the first block is on the second block.
