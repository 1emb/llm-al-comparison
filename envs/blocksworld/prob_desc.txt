This domain models a simple blocks-manipulation world in which blocks can rest on the table, be stacked on other blocks, be clear on top, and be held by a single hand that is either empty or holding exactly one block. A block is considered to be on another block when it is stacked directly on it, and a block is considered to be on the table when it is resting directly on the table; a block is clear when there is nothing on top of it; and the hand is empty when it is not holding any block. There are four kinds of actions. Picking up a block can happen only when that block is clear, is on the table, and the hand is empty; after picking it up, the block is no longer on the table, it is no longer clear, the hand is no longer empty, and the hand is holding that block. Putting down a block can happen only when the hand is holding that block; after putting it down, the hand is no longer holding it, the block becomes clear, the hand becomes empty, and the block is on the table. Stacking a block onto another block can happen only when the hand is holding the first block and the second block is clear; after stacking, the hand is no longer holding the first block, the second block is no longer clear, the first block becomes clear, the hand becomes empty, and the first block is on top of the second block. Unstacking a block from on top of another can happen only when the first block is on the second block, the first block is clear, and the hand is empty; after unstacking, the hand is holding the first block, the second block becomes clear, the first block is no longer clear, the hand is no longer empty, and the first block is no longer on the second block.
