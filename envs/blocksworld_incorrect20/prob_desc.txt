This domain describes a world of items that can rest on the table, be placed on top of one another, be clear on top, and be either held or not, with a condition indicating whether the hand is empty. The pick-up action is used to take an item from the table, and it can occur only when the item is clear, the item is on the table, and it is not the case that the hand is empty; after performing it, the item is no longer on the table, the item is no longer clear, the hand is no longer empty, and it is no longer the case that the item is being held. The put-down action places an item onto the table, and it can occur only when it is not the case that the item is being held; after performing it, it is no longer the case that the item is being held, the item becomes clear, the hand is no longer empty, and the item is on the table. The stack action places a held item onto another item, and it can occur only when the first item is being held and it is not the case that the second item is clear; after performing it, the first item is no longer being held, the second item becomes clear, the first item becomes clear, the hand becomes empty, and the first item ends up on the second item. The unstack action removes an item from on top of another item, and it can occur only when the first item is on the second item, it is not the case that the first item is clear, and the hand is empty; after performing it, the first item is being held, the second item becomes clear, the first item is no longer clear, the hand becomes empty, and the first item is no longer on the second item.
