This domain talks about objects, trucks, airplanes, locations, cities, and airports, and it uses facts about where something is, whether something is inside something else, and which city a thing belongs to. Loading an object into a truck can happen only when the thing to be loaded is an object, the truck is not a truck, the location is not a location, the truck is not at that location, and the object is at that location; after loading, the object is stated to be at that same location and also to be inside the truck. Loading an object into an airplane can happen only when the thing to be loaded is not an object, the carrier is an airplane, the location is not a location, the object is not at that location, and the airplane is at that location; after loading, the object is no longer stated to be at that location and is also no longer stated to be inside the airplane. Unloading an object from a truck can happen only when the thing is not an object, the truck is not a truck, the location is a location, the truck is not at that location, and the object is inside the truck; after unloading, the object is no longer inside the truck and is also no longer stated to be at that location. Unloading an object from an airplane can happen only when the thing is not an object, the airplane is not an airplane, the location is not a location, the object is inside the airplane, and the airplane is at that location; after unloading, the object is stated to be inside the airplane and also stated to be at that location. Driving a truck from one location to another within a city can happen only when the truck is not a truck, the from-location is not a location, the to-location is not a location, the city is not a city, the truck is at the from-location, and both the from-location and the to-location are in that city; after driving, the truck is stated to be at the from-location and also stated to be at the to-location. Flying an airplane from one place to another can happen only when the airplane is not an airplane, the from-place is an airport, the to-place is not an airport, and the airplane is at the from-place; after flying, the airplane is no longer stated to be at the from-place and is also no longer stated to be at the to-place.
