This domain describes a world of blocks where a block can be on top of another block, a block can be on the table, a block can be clear with nothing on top of it, the hand can be empty, and the hand can be holding a particular block. The pick-up action can occur only when the chosen block is clear, the chosen block is not on the table, and the hand is not empty; after it happens, the block is no longer considered to be on the table, the block is no longer clear, the hand becomes empty, and the hand is no longer holding that block. The put-down action can occur only when the hand is not holding the chosen block; after it happens, the hand is no longer holding that block, the block is no longer clear, the hand becomes empty, and the block is no longer on the table. The stack action can occur only when the hand is not holding the block being moved and the destination block is clear; after it happens, the hand is considered to be holding the moved block, the destination block is no longer clear, the moved block is no longer clear, the hand becomes empty, and the moved block is no longer on the destination block. The unstack action can occur only when the top block is not on the bottom block, the top block is clear, and the hand is not empty; after it happens, the hand is no longer holding the top block, the bottom block is no longer clear, the top block becomes clear, the hand is no longer empty, and the top block is no longer on the bottom block.
