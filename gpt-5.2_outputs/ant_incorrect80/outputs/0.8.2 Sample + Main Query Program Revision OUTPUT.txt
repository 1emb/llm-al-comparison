% Original meaning: ant1 at p1; p1 adjacent p2 and p2 adjacent p1; p1 clear and p2 clear;
% p1 is the nest; food at p2; ant has no food. Find a plan to bring food to the nest.
%
% Fix: If isClear(p2) then move to p2 is forbidden by rule (5), and if isClear(p1) then
% clearObstacle cannot make p1/p2 "not clear" unless they are already clear (rule 19),
% but there is no action to make a place clear. So the original initial state makes
% the task impossible under this program.
%
% To preserve the intended "bring food from p2 to nest p1" task under the given program,
% we keep adjacency/nest/food/loc/hasFood, but set ~isClear(p2) so moving to p2 is allowed.
:- query
    0: loc(ant1)=p1 &
       adjacent(p1,p2) & adjacent(p2,p1) &
       isNest(p1) &
       hasFoodAt(p2) &
       ~hasFood(ant1) &
       ~isClear(p2);
    maxstep: hasFoodAt(p1).
