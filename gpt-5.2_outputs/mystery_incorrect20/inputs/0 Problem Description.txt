This domain talks about objects that may be provinces or planets, a global condition called harmony that is either present or absent, an object-level condition of being in pain, and a relation where one object craves another. The attack action can be performed on an object only if that object is both a province and a planet and harmony is currently absent; after the attack, that object is made to be in pain, it stops being a province and stops being a planet, and harmony becomes present. The succumb action can be performed on an object only if that object is not in pain; after succumbing, the object becomes a province and harmony becomes present, while the object stops being a planet, and its pain status is removed. The overcome action involves two objects and can be performed only if the second object is a province and the first object is not in pain; after overcoming, harmony becomes present, the first object stops being a province, the first object comes to crave the second, the second object stops being a province, and any pain status on the first object is removed. The feast action also involves two objects and can be performed only if the first object craves the second, the first object is not a province, and harmony is present; after feasting, the first object is put into pain, the second object becomes a province, the craving relation from the first to the second is removed, the first object becomes a province, and harmony is removed.
