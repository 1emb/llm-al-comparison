This domain models windows that may be described as open or closed and as locked or unlocked, but all actions are only applicable to something that is explicitly not a window. The action to open a window can occur only when the item is not a window, it is closed, and it is not unlocked; after this action, the item is made closed and it is made not open. The action to close a window can occur only when the item is not a window and it is not open; after this action, the item is made open and it is made not closed. The action to lock a window can occur only when the item is not a window, it is closed, and it is not unlocked; after this action, the item is made unlocked and it is made not locked. The action to unlock a window can occur only when the item is not a window and it is not locked; after this action, the item is made locked and it is made not unlocked.
