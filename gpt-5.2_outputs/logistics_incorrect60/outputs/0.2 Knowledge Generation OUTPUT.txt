% If a package is inside a vehicle, then the package is at the same location as that vehicle.
% If a package is inside a vehicle, then the package is not directly at any location different from the vehicle's location.

% Loading a package into a truck at a location causes the package to be inside that truck.
% Loading a package into a truck at a location causes the package to be at that location.

% Loading a package into a truck at a location is not executable if the package is not at that location.
% Loading a package into a truck at a location is not executable if the truck is not at that location.
% Loading a package into a truck at a location is not executable if the package is already inside some vehicle.

% Loading a package into an airplane at a location causes the package to be inside that airplane.
% Loading a package into an airplane at a location causes the package to not be directly at that location.

% Loading a package into an airplane at a location is not executable if the airplane is not at that location.
% Loading a package into an airplane at a location is not executable if the package is already inside some vehicle.

% Unloading a package from a truck at a location causes the package to not be inside that truck.
% Unloading a package from a truck at a location causes the package to be at that location.

% Unloading a package from a truck at a location is not executable if the truck is not at that location.
% Unloading a package from a truck at a location is not executable if the package is not inside that truck.

% Unloading a package from an airplane at a location causes the package to not be inside that airplane.
% Unloading a package from an airplane at a location causes the package to be at that location.

% Unloading a package from an airplane at a location is not executable if the airplane is not at that location.
% Unloading a package from an airplane at a location is not executable if the package is not inside that airplane.

% Driving a truck from one location to another in a city causes the truck to be at the to-location.
% Driving a truck from one location to another in a city causes the truck to not be at the from-location.

% Driving a truck from one location to another in a city is not executable if the truck is not at the from-location.
% Driving a truck from one location to another in a city is not executable if the from-location is not in that city.
% Driving a truck from one location to another in a city is not executable if the to-location is not in that city.
% Driving a truck from one location to another in a city is not executable if the from-location is the same as the to-location.

% Flying an airplane from one location to another causes the airplane to be at the to-location.
% Flying an airplane from one location to another causes the airplane to not be at the from-location.

% Flying an airplane from one location to another is not executable if the airplane is not at the from-location.
% Flying an airplane from one location to another is not executable if the from-location is not an airport.
% Flying an airplane from one location to another is not executable if the to-location is not an airport.
% Flying an airplane from one location to another is not executable if the from-location is the same as the to-location.
