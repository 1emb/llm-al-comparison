This domain talks about objects, trucks, airplanes, locations, cities, and airports, and it can represent that something is at a location, that something is inside a vehicle, and that an object or location is associated with a city. Loading an object into a truck can happen only if it is not the case that the thing being loaded counts as an object, it is not the case that the vehicle counts as a truck, it is not the case that the place counts as a location, the truck is not at that place, and the object is at that place; after this loading happens, the object is made to be at that same place, and it is removed from being inside the truck. Loading an object into an airplane can happen only if it is not the case that the thing being loaded counts as an object, the vehicle does count as an airplane, it is not the case that the place counts as a location, the object is not at that place, and the airplane is not at that place; after this loading happens, the object is made to be at that place, and it is removed from being inside the airplane. Unloading an object from a truck at a place can happen only if it is not the case that the thing being unloaded counts as an object, it is not the case that the vehicle counts as a truck, it is not the case that the place counts as a location, the truck is at that place, and the object is not inside the truck; after this unloading happens, the object is made to be inside the truck, and it is removed from being at that place. Unloading an object from an airplane at a place can happen only if it is not the case that the thing being unloaded counts as an object, the vehicle does count as an airplane, it is not the case that the place counts as a location, the object is not inside the airplane, and the airplane is not at that place; after this unloading happens, the object is made to be inside the airplane, and it is removed from being at that place. Driving a truck from one place to another within a city can happen only if it is not the case that the vehicle counts as a truck, it is not the case that the starting place counts as a location, the destination does count as a location, it is not the case that the city counts as a city, the truck is not at the starting place, the starting place is not in that city, and the destination is not in that city; after driving, the truck is made to be at the starting place and is removed from being at the destination. Flying an airplane from one airport to another can happen only if it is not the case that the vehicle counts as an airplane, it is not the case that the starting place counts as an airport, the destination does count as an airport, and the airplane is not at the starting place; after flying, the airplane is made to be at the starting place and is removed from being at the destination.
