This domain models objects, trucks, airplanes, locations, cities, and airports, along with facts about where things are, what is inside what, and which city something belongs to. The action of loading an object into a truck can occur only if the thing being loaded is an object, the carrier is a truck, the specified place is not a location, the truck is not at that place, and the object is at that place; after this action, the object is no longer at that place and it is also no longer inside the truck. Loading an object into an airplane can occur only if the thing being loaded is an object, the carrier is not an airplane, the specified place is a location, the object is not at that place, and the airplane is at that place; after this action, the object is no longer at that place and it is also no longer inside the airplane. Unloading an object from a truck at a place can occur only if the thing being unloaded is not an object, the carrier is a truck, the specified place is not a location, the truck is at that place, and the object is inside the truck; after this action, the object is inside the truck and the object is at that place. Unloading an object from an airplane at a place can occur only if the thing being unloaded is not an object, the carrier is an airplane, the specified place is a location, the object is inside the airplane, and the airplane is not at that place; after this action, the object is inside the airplane and the object is at that place. Driving a truck from one place to another within a city can occur only if the vehicle is not a truck, the starting place is a location, the destination place is not a location, the city is a city, the truck is at the starting place, the starting place is in the given city, and the destination place is not in that city; after driving, the truck is no longer at the starting place and it is also no longer at the destination place. Flying an airplane from one place to another can occur only if the vehicle is not an airplane, the starting place is not an airport, the destination place is an airport, and the airplane is at the starting place; after flying, the airplane is at the starting place and it is also at the destination place.
