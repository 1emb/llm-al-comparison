There are five types of objects: "package", "truck", "airplane", "location", and "city".
"vehicle" is a common type for "truck" and "airplane".
"thing" is a common type for "package" and "vehicle".

Unloading a package from an airplane at a location is an action.
Driving a truck from a location to a location in a city is an action.
